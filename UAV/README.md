# 🚀MF-UAV

> 这是一个关于`cesium`框架的一个无人机模拟控制器的小`demo`

![screenshot](https://github.com/fengtianxi001/MF-UAV/blob/master/screenshot/index.gif?raw=true)


## 1. 项目功能点

- 通过按键调整飞行器的飞行姿态

- 按键无冲

- 实时获取飞行器的飞行参数

- 通过飞行器的经纬度坐标完成leaflet的轨迹定位

## 2. 启动项目

`npm install & npm run dev`
